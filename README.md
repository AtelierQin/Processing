# 三峡大坝等高线图模拟

## 项目介绍
这是一个使用Processing语言创建的三峡大坝等高线图模拟程序。该程序通过柏林噪声(Perlin noise)算法生成地形高度数据，并使用颜色渐变表示不同高度的等高线，从而模拟三峡大坝及其周围地形的视觉效果。

## 功能特点
- 使用柏林噪声算法生成自然地形
- 模拟三峡大坝结构和水位差
- 使用颜色渐变表示不同高度的等高线
- 交互功能：鼠标拖动可以改变地形生成参数
- 按键控制：按'R'键可以重新生成地形

## 使用方法
1. 安装Processing开发环境（可从[Processing官网](https://processing.org/download/)下载）
2. 打开ThreeGorgesDamContour.pde文件
3. 点击运行按钮执行程序
4. 使用鼠标拖动可以改变地形生成参数
5. 按'R'键可以重新生成地形

## 技术说明
- 地形生成：使用柏林噪声算法创建自然地形
- 等高线绘制：根据高度值绘制等高线
- 颜色映射：
  - 蓝色：表示水域
  - 绿色：表示低地
  - 棕色到白色：表示高地
- 3D渲染：使用Processing的P3D渲染器实现三维效果

## 自定义参数
如果想调整模拟效果，可以修改以下参数：
- `scl`：地形网格大小
- `noiseScale`：噪声缩放因子，影响地形的粗糙度
- `waterLevel`：水位高度
- 大坝位置和尺寸：`damX`, `damY`, `damWidth`, `damHeight`

## 注意事项
- 程序运行需要一定的计算资源，如果运行缓慢，可以尝试减小画布大小或增大网格大小
- 该模拟为艺术创作，不代表真实的三峡大坝地形
